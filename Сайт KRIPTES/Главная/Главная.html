Что такое ит-клуб?:
«ИТ Клуб» — это инструмент PR-специалиста для общения с целевой аудиторией.


В чем наша цель?

Цель Kriptes-научить детей сфере IT так, как не знают даже некоторые взрослые люди. Наш IT клуб поможет юным программистам познать все в мире программирования и обучат таким языкам программирования, как: CSS, JavaSkript,HTML,С++ и др.

Чему обучает организация Kriptes?

В нашей школе имеется множество кружков для внеурочных занятий. Из них довольно много кружков в IT сфере. Arduino,Веб-дизайн,Робототехника и это всё только верхушка айсберга! По мимо них имеются и другие кружки.

Что Представляет Собой Web-Design?

Веб-дизайн-вид программирования использующийся для создания сайтов и веб-страниц. В веб-дизайне используются языки HTML,CSS и JavaScript. JavaScript используется для создания более продвинутых сайтов, однако его необязательно использовать, он нужен лишь для усовершенствования технологической части сайта. HTML основной язык, отвечает за "скелет" сайта. CSS используется для дизайнерской части сайта, то есть для косметической.
